# O que é API REST

API (Application Programming Interface) é um conjunto de regras que permite
que sistemas se comuniquem entre si.

REST é um padrão arquitetural que utiliza o protocolo HTTP para realizar
operações como buscar, criar, atualizar e deletar dados.

Uma API REST geralmente trabalha com:

- Cliente (quem faz a requisição)
- Servidor (quem responde)
- Métodos HTTP (GET, POST, PUT, DELETE)
- Dados no formato JSON

## Papel do QA

O QA testa se:

- A API retorna o status code correto
- Os dados retornados estão corretos
- As validações estão funcionando
- Não há falhas ou inconsistências
